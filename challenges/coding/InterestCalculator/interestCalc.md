# 🚀 Simple Interest Calculator

## 📝 Task Description
A Simple Interest (SI) Calculator takes 3 inputs: 
- Principal Amount (P)
- Rate of Interest (R)
- Time Duration (T)

The SI is calculated as
SI = (P * R * T) / 100

## 📤 Sample Output
```
Principal Amount: 5000
Rate of Interest: 10
Time Duration: 2
Simple Interest is: Rs. 1000 
```
